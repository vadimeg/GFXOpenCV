Манаул для сборки проекта под Windows (VS 2013):
================================================
1) Клонировать репозиторий оригинального OpenCV	
1*) Для полной совместимости исходников, необходимо проделать несколько операций: 
	> Зайти из под консоли с Git окружением в папку с проектом
	> Выполнить команду git checkout b0b1943
2) Далее необходимо загрузить утилиту CMake (версии выше 3.4.0)
3) Клонировать данный репозиторий в папку с уже клонированным OpenCV
4) Запустить скрипт Build.bat
5) Результат сборки можно будет найти в папке Build 
(папка "Build" будет лежать в директории из которой был вызван скрипт)
