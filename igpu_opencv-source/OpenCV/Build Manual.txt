Манаул для сборки проекта под Windows (VS 2013):
================================================
1) Клонировать репозиторий оригинального OpenCV	(версия 3.1 на данный момент)
2) Далее необходимо загрузить утилиту CMake (версии выше 3.4.0)
3) Клонировать данный репозиторий в папку с уже клонированным OpenCV
4) Скопировать файлы из папки source_diff в корневую папку с opencv
5) На одном уровне с папкой "opencv" положить скрипт Build.bat (*)
5) Запустить скрипт Build.bat из под Intel Visual Studio Environment c параметрами:
	1-ый параметр: Разрядность бибилотеки (x86, x64)
	2-ой параметр: Путь до папки "opencv"

	(По умолчанию, если не задавать перед запуском скрипт никаких аргументов, 
	сборка будет производиться под 64-разрядную машину, а папка по умолчанию
	будет "opencv")
6) Результат сборки можно будет найти в папке Build/*тип сборки*/bin/Release/ 
(папка "Build" будет лежать в директории из которой был вызван скрипт)

(*) - необходимая иерархия
--------
opencv
Build.bat
