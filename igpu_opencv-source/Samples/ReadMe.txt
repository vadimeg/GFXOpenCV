Запуск тестов производить из текущей директорий (запустить файл testInvoker.bat). Для запуска последовательного тестирования всех фильтров, необходимо в текущую папку положить интересующую версию библиотеку OpenCV, а именно файлы (где * - номер версии библиотеки (данные примеры компилировались с версией OpenCV 3.0)):
	> opencv_core*.dll
	> opencv_imgproc*.dll
	> opencv_imgcodecs*.dll

В случае с Билатеральным фильтром, перед запуском тестирования, необходимо выполнить инструкцию в файле ReadMe, каталога Bilateral. Если возникают проблемы с остальными фильтрами, необходимо перекомпилировать их на текущей машине. Исходные коды тестов лежат в соответствующих каталогах. 

Компиляция производится запуском скрипта testBuilder.bat (запуск также производить из текущей директории). В текущей директории, необходимо создать папку OpenCV, в ней создать каталоги include и lib соответственно. В эти каталоги необходимо положить каталоги с include файлами собранной OpenCV, а в каталог lib соответственно положить .lib файлы собранной OpenCV. Файл testBuilder.bat запускать из под Intel 64 Visual Studio environment

При компиляции соответственно использовать версию OpenCV >= 3.0. Более подробная информация по компиляции, и сборке OpenCV лежит в каталоге OpenCV (находится выше на один уровень).
